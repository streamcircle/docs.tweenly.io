# Release notes - Tweenly Editor 2.1.4

### New Features

**Expand / Collapse All in Objects Tree**
New option to expand or collapse all objects in the tree at once — saves time when working with complex scenes.

**Animations Sorted by Start Time**
Animations are now sorted by start time in the list, making it easier to navigate the project timeline.

---

### Improvements

**New Fonts: Manrope & Inter**
The editor now uses Manrope and Inter fonts for more consistent and readable typography across the entire interface.

**Performance Improvements**
Optimized internal updates when working with object properties — the editor responds more smoothly.

---

### Bug Fixes

- **Playback:** The "Next" button at position 0 now correctly starts playback instead of jumping to the next marker.
- **Timeline:** Fixed inverted expand/collapse logic for timeline rows.
- **Fonts:** Manrope font now loads correctly in both the web and graphics-editor environments.
- **Canvas:** Disabled double-click on the canvas, which was causing unintended actions.
